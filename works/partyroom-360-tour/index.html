PROJECT GOAL

예쁜 사진을 넘어서
방문 전 판단을 돕는 페이지.

파티룸 고객은 예약 전에 좌석 수, PC 배치, TV와 게임기, 주방 시설, 공간 사이 동선을 궁금해합니다. 그래서 사진을 단순 나열하지 않고, 전체 구조는 360° 투어로, 시설 디테일은 고화질 사진으로, 사용법은 안내 가이드로 나눠 전달했습니다.

같은 촬영 결과를 LavaLabs 포트폴리오와 실제 파티룸 고객용 사이트에 각각 다른 목적과 문장으로 재구성했습니다.

INTERACTIVE 360 TOUR

HIGH-RES PHOTOGRAPHY

작은 썸네일이 아니라
카메라 원본을 제대로 사용했습니다.

보정된 카메라 원본을 웹용 고화질 이미지로 다시 제작하고, 가로·세로 사진의 원본 비율에 맞춰 배치했습니다.

SOLUTION PACKAGE

촬영 결과를 실제 영업 자산으로 바꿉니다.

01

고화질 공간 사진

대표 이미지와 구역별 상세 사진을 웹, 플레이스와 SNS에 재사용할 수 있게 구성합니다.

02

360° 가상투어

여러 촬영 지점을 연결해 고객이 공간을 직접 이동하며 확인하게 합니다.

03

도면 기반 미니맵

현재 위치와 공간 구역을 동시에 보여주어 360 화면의 방향 감각을 보완합니다.

04

실사용 웹사이트

시설, 이용수칙과 안내책자 내용을 고객이 읽기 쉬운 모바일 페이지로 정리합니다.

WORKFLOW

현장 기록부터 공개 링크까지.

  1. 현장 구조 확인

    시설, 가구 배치와 고객이 궁금해할 지점을 기록합니다.

  2. 일반·360 촬영

    전체 공간과 시설 디테일을 용도에 맞는 방식으로 촬영합니다.

  3. 이미지 최적화

    고해상도 원본에서 반응형 웹 이미지를 다시 제작합니다.

  4. 투어·도면 구현

    장면 연결, 이동 버튼과 위치 동기화를 구성합니다.

  5. 사이트 제작

    소개, 사진, 투어, 이용 안내와 문의 동선을 정리합니다.

LIVE CUSTOMER SITE

포트폴리오와 별도로
실제 고객용 사이트를 제작했습니다.
